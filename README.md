Kids Vaccination Booking System (KVBS)

📌 Overview
The Kids Vaccination Booking System is a web-based application developed using PHP and MySQL. It allows parents to schedule vaccination appointments for their children efficiently and helps administrators manage bookings.

🚀 Features
- User registration & login system
- Vaccination appointment booking
- Parents can manage booking
- Parents can give vaccination feedback
- Parents can download vaccination certificates after successfull vaccination 
- View booking history
- Admin panel for management
- Manage users & appointments

🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Server: XAMPP / WAMP
